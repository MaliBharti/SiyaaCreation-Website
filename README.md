# Siyaa Creation – E-commerce Website

## Project Overview
Siyaa Creation is an e-commerce web application developed using Java Servlets and JSP. 
The project allows users to browse products, create an account, log in, add items to the cart, and place orders. 
It focuses on implementing basic e-commerce functionality along with a structured backend.

## Features
* User registration and login
* Product listing
* Add to cart and remove from cart
* Order placement
* Session handling for login/logout

## Tech Stack
Frontend: HTML, CSS, JavaScript, JSP
Backend: Java Servlets
Database: MySQL
Server: Apache Tomcat

## Live Demo
